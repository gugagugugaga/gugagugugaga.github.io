核心公式说明

Φ空间色差（动态权重）：
φL(L) = L + 100·tanh(0.0152·(L−50)) − 1.6014·(L−50) + 0.0509
φC(C) = 42.381·ln(1 + 0.0734·C)
φh(h,C) = h + f(C)·[G_f(h) + G_p(h)]
wL = 0.8579 + 0.9992·e^(−2.6811·|ΔφL|)（亮度权重动态衰减）
wC = 0.4290 + 1.9097·e^(−5.3792·|ΔφC|)（彩度权重动态衰减）
wH = 0.0265 + 0.4448·e^(−0.0085·|ΔφH|)（色相权重动态衰减）
ΔE = √[(wL·ΔφL)² + (wC·ΔφC)² + (wH·ΔφH)²]

CMC(l:c)： ΔE = √[(ΔL/2S_L)² + (ΔC/S_C)² + (ΔH/S_H)²]，权重基于标样计算
CIEDE2000： 标准加权欧氏距离 + 旋转修正项 R_T
