核心公式

Φ空间色差：
φL(L) = L + 100·tanh(0.0152(L-50)) - 1.6014(L-50) + 0.0509
φC(C) = 42.381·ln(1 + 0.0734·C)
φh(h,C) = h + f(C)·[G_f(h) + G_p(h)]
ΔE = ΔE_base · scale(ΔE_base),   scale(x) = 1.0736 + 1.1942·e^(-1.8467x)

CMC(l:c)： ΔE = sqrt[(ΔL/2S_L)^2 + (ΔC/S_C)^2 + (ΔH/S_H)^2]
CIEDE2000： 标准加权欧氏距离 + 旋转修正项
